# 社区物业管理系统 java151

## 项目概述

社区物业管理系统是基于 Spring Boot 的全功能物业管理系统。该系统采用前后端分离的设计模式，前端使用 Vue 框架，后端使用 Spring Boot 框架，内置数据库文件，便于直接部署。

## 技术栈

- 前端：Vue 框架
- 后端：Spring Boot 框架

## 功能模块

### 管理员模块

- **社区管理**
  - 基本信息 management
  - 周边设施 management
  - 物业公告 management
- **楼盘管理**
  - 楼宇 management
  - 房间 management
- **社区保障管理**
  - 报修 management
  - 投诉 management
- **收费管理**
  - 收费类型 management

### 用户模块

- **基本信息管理**
  - 个人信息修改
  - 密码修改
- **物业收费**
  - 缴纳费用
- **报修管理**
  - 申请报修
  - 报修管理
- **投诉管理**
  - 发起投诉
  - 投诉管理

## 系统角色

- 管理员
- 住户

## 运行环境

- 前端：现代浏览器
- 后端：Java 1.8 或以上版本环境

## 部署步骤

1. 准备所需环境（如 Java、数据库等）
2. 部署后端服务
3. 部署前端服务
4. 启动服务

